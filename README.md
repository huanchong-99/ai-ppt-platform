# AI PPT Platform

一个基于AI的PPT创作平台，帮助用户快速生成高质量的演示文稿。

## 功能特点

- AI辅助内容生成
- 多种精美模板
- 实时编辑与预览
- 一键导出
- 团队协作

## 技术栈

- Next.js 15
- React 19
- TypeScript
- Redux
- Ant Design
- Tailwind CSS
- Styled-components
- Axios

## 开发环境搭建

```bash
# 安装依赖
npm install --legacy-peer-deps

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm run start
```

## 项目结构

- `/app` - Next.js页面与路由
- `/components` - 可复用组件
- `/hooks` - 自定义React Hooks
- `/lib` - 工具函数和辅助库
- `/public` - 静态资源
- `/styles` - 全局样式
- `/src` - 源代码
